Для того чтобы при нажатии на определенный жанр отображался список игр, вам нужно сделать следующее:

Создайте представление Django, которое будет обрабатывать запросы на отображение списка игр для определенного жанра. 
В этом представлении вы должны получить список игр, относящихся к выбранному жанру, и передать его в контекст шаблона.

В вашем шаблоне HTML, где отображается список жанров (main-menu), добавьте ссылки для каждого жанра. Эти ссылки должны вести на URL, 
который будет обрабатывать запросы для отображения списка игр по определенному жанру. Например:

html
Copy code
<ul class="main-menu">
    <li class="menu-item"><a href="{% url 'shop_app:games_by_genre' 'Action' %}">Action</a></li>
    <li class="menu-item"><a href="{% url 'shop_app:games_by_genre' 'RPG' %}">RPG</a></li>
    <!-- Добавьте ссылки для других жанров по аналогии -->
</ul>
----------------------------------------------------------------------------------------------------------------------------------------------
Создайте соответствующий URL-шаблон, который будет маршрутизировать запросы на отображение списка игр по жанру. Например:


# urls.py
from django.urls import path
from . import views

app_name = "shop_app"

urlpatterns = [
    # ... другие URL-шаблоны ...

    path("games-by-genre/<str:genre_name>/", views.games_by_genre, name="games_by_genre"),
]
---------------------------------------------------------------------------------------------------------------------------------------
В представлении, которое обрабатывает запросы для отображения списка игр по жанру, получите выбранный жанр и получите список игр, 
относящихся к этому жанру, из базы данных. Затем передайте этот список в контекст шаблона и отобразите его.

# views.py
from django.shortcuts import render
from .models import Game

def games_by_genre(request, genre_name):
    games = Game.objects.filter(genre=genre_name)
    return render(request, 'shop_app/games_by_genre.html', {'games': games})

Создайте шаблон games_by_genre.html, который будет отображать список игр для выбранного жанра. В этом шаблоне вы можете использовать 
цикл для отображения списка игр.
Теперь, при нажатии на определенный жанр, вы будете перенаправлены на страницу со списком игр для этого жанра.
